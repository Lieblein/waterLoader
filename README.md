## npm команды
- `yarn run start` - для разработки
- `yarn run build` - сборка билда
- `yarn run lint` - проверка кодстайла `js и css`
- `yarn run lint-js` - проверка кодстайла `только js`
- `yarn run lint-css` - проверка кодстайла `только css`
